# 应用配置文件

项目可能会用到的配置文件的格式

## TOML

> Tom's Obvious, Minimal Language

- 文件后缀为`.tml`

## YAML

> YAML Ain't Markup Language

- 文件后缀为`.yml`
- 使用缩进和换行表示层次结构

```yaml
# 注释
键: 值
键: 值
```

## INI

INI 是一种用于存储简单配置信息的文件格式

- 文件后缀为`.ini`
- 文件内由字节 ( section )、键值对 ( key-value-pair ) 组成

```ini
; 注释
[字节]
键=值
键=值
```

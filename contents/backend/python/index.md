## 安装

::: tip
本文使用 [asdf](/others/tools/asdf/)
:::

::: code-group

```shell [安装]
# 1.
asdf plugin add python
# 2.
asdf install python [版本]
# 3.
asdf global python [版本]
# 4.
asdf reshim python
```

```shell [版本查看]
python --version
```

:::

> 本文使用 v3.10.0

```shell
% python --version
Python 3.10.0
```

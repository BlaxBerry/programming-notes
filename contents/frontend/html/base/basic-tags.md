## 标题标签

### \<h1> ~ \<h6>

> 块级元素

```html
<h1>h1 标题</h1>
<h2>h2 标题</h2>
<h3>h3 标题</h3>
<h4>h4 标题</h4>
<h5>h5 标题</h5>
<h6>h6 标题</h6>
```

## 段落标签

### \<p>

> 块级元素

`<p>`标签内不能包含另一个`<p>`标签，否则报错

```html
<p>内容内容</p>
```

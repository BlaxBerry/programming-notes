# JS 基本数据类型

基于数据的性质与结构可分为：

- 基本类型数据 ( Primitive Types )
- [引用类型数据](./reference-data-type.md) ( Reference Types )

## 字符串类型 ( string )

```js
const 字符串对象 = String();
```

## 数字类型 ( number )

表示整数或浮点数

## 布尔类型 ( boolean )

```js
const 布尔对象 = Boolean();
```

## undefined

表示未定义的值

## null

表示空值或者没有对象值

## Symbol

ES6 新增

```js
const Symbol对象 = Symbol();
```
